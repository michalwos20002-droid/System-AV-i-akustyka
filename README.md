# Projekt audiowizualnego systemu prezentacji dla sali konferencyjnej

Kompleksowe opracowanie inżynierskie systemu multimedialnego, akustycznego i nagłośnieniowego dla dużej sali prezentacyjnej o kubaturze 714 m3.

## 🛠️ Narzędzia i standardy
* **Obliczenia akustyczne:** Wzór Sabine'a, Rigips Acoustic Calculator, mody własne i rezonanse pomieszczenia
* **Technologie AV:** Systemy nagłośnienia radiowęzłowego 100V, optyka i strumień świetlny projektorów (ANSI lumeny, throw ratio, foot-lamberts)
* **Normy i biznes:** Norma akustyczna PN-B-02151-4, kosztorysowanie inwestorskie

## 📋 Zakres obliczeń inżynierskich
1. **Analiza akustyczna:** Obliczenie dokładnego czasu pogłosu T60 oraz zaprojektowanie optymalnej adaptacji akustycznej ścian i sufitów zgodnie z polskimi normami budowlanymi.
2. **Elektroakustyka:** Wyznaczenie stref pokrycia dźwiękiem dla głośników sufitowych oraz obliczenie wymaganego zapotrzebowania na moc wzmacniacza w instalacji 100V.
3. **System wideo:** Kalkulacja parametrów optycznych projektora biznesowego, obliczenie współczynnika proporcji rzutu (throw ratio) oraz jasności ekranu dla zapewnienia idealnej widoczności w każdych warunkach oświetleniowych.
4. **Kosztorys:** Przygotowanie kompletnego spisu sprzętu wraz z rynkowym kosztorysem inwestorskim zamykającym się w kwocie ponad 19 tys. zł.
